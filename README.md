# Troy Groups Website

A modern, responsive website for Troy Groups - a global leader in outsourced capital raising and direct investment opportunities.

## Features

- Responsive design built with Bootstrap 5
- Interactive components and animations
- Modern CSS with custom styling
- Multiple service pages and contact forms
- About page with team information and success stories

## Structure

- `index.html` - Homepage
- `about.html` - About page with team and company information
- `services.html` - Services overview
- `contact.html` - Contact information and forms
- `assets/` - CSS, JavaScript, images, and other assets
- `Components/` - Reusable HTML components

## Technologies Used

- HTML5
- CSS3 with custom properties
- JavaScript (ES6+)
- Bootstrap 5.3.3
- Bootstrap Icons
- Font Awesome
- Google Fonts (Inter, Montserrat)

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser or serve via a local web server
3. No build process required - static HTML/CSS/JS

## Development

The website uses modern CSS features including:
- CSS Grid and Flexbox
- Custom properties (CSS variables)
- Responsive design with mobile-first approach
- Smooth animations and transitions
